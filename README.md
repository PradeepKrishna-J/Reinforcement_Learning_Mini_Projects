# Reinforcement Learning Mini Projects

A collection of educational Reinforcement Learning implementations using various algorithms and environments. These projects demonstrate fundamental RL concepts and algorithms through practical examples.

## üìö Projects Overview

### Grid World & Navigation
- **5X5 Grid World (Optimal Policy)** - Finding optimal policies in a 5x5 grid environment
- **Frozen Lake Q-Learning** - Q-learning implementation on the Frozen Lake environment
- **Frozen Lake SARSA** - SARSA algorithm applied to Frozen Lake
- **Frozen Lake (Basic)** - Introductory Frozen Lake exploration
- **Cliff Walking (Off-Policy Q-Learning)** - Off-policy learning in cliff walking environment
- **Value Iteration Grid World** - Value iteration implementation for grid world

### Classic RL Problems
- **Gambler's Problem** - Dynamic programming solution to the gambler's problem
- **Jack's Car Rental (Policy Iteration)** - Policy iteration for the car rental problem
- **Jack's Car Rental (Value Iteration)** - Value iteration approach to car rental
- **Recycling Robot** - MDP modeling and solution for a recycling robot

### Card Games & Strategy
- **BlackJack Monte Carlo** - Monte Carlo methods for BlackJack strategy
- **Tic-Tac-Toe (Exercise 5a)** - RL approach to Tic-Tac-Toe
- **Tic-Tac-Toe Q-Learning** - Q-learning implementation for Tic-Tac-Toe

### Advanced Environments
- **Taxi Pick-Drop** - Q-learning for the Taxi environment

### Theoretical Foundations
- **Markov Process** - Introduction to Markov processes
- **Value Iteration and Policy Iteration Algorithms** - Comprehensive comparison of DP methods

## üöÄ Getting Started

### Prerequisites
```bash
pip install gymnasium numpy matplotlib jupyter
```

### Running the Projects
1. Clone this repository
2. Navigate to the project directory
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Open any `.ipynb` file to explore the implementation

## üìñ Learning Path

**Recommended order for beginners:**
1. Start with `Markov_process.ipynb` for theoretical foundations
2. Explore `value_iteration_and_policy_iteration_algorithms.ipynb`
3. Try simpler environments: `frozen_lake.ipynb`
4. Progress to more complex problems: `Jacks'_Car_Rental_*.ipynb`
5. Experiment with different algorithms on the same environment
